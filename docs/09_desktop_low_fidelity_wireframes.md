# Desktop Low-Fidelity Wireframes

Last reviewed: 2026-04-15

## Purpose

These wireframes describe the main desktop surfaces at a structural level. They are not pixel-perfect mocks.

## 1. Hatching entry

```text
+-----------------------------------------------------------+
|                   mascot / product identity               |
|                                                           |
|                 Create your AzulClaw companion            |
|         Workspace, memory, personality, and safety        |
|                                                           |
|                 [ Start hatching ]                        |
|                 [ Restore profile ]                       |
+-----------------------------------------------------------+
```
