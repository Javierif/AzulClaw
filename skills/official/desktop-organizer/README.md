# Folder Organizer Skill

First-party local MCP skill for browsing and organizing files inside a
user-selected folder tree.

It can:
- list folders and files inside the granted root folder
- preview an organization plan for the root or a relative subfolder
- build a global plan for a subtree and move matching files into shared category folders under that subtree root
- limit how deep it scans with the configured traversal depth or a per-request override
- optionally let the agent propose custom folder names when semantic categorization is enabled in the skill config

It does not delete files.
